# Telegram Bot
### Telegram Bot using Node.js tutorial

This is the source code for **How to build a Telegram Bot using Node.js** tutorial from my blog **https://thinkcodeplay.com**

See full tutorial [here]



## Installation

App requires [Node.js](https://nodejs.org/) run.

Install the dependencies and devDependencies and start the server:

```sh
cd telegram-bot-gamedeal-node.js
npm install
npm run dev
```
